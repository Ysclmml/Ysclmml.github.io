基于freerdp远程连接

前言

因工作需要, 需要使用ubuntu连接远程Windows电脑. 在windows电脑上使用向日葵等远程连接软件十分方便, 在ubuntu上当然也有对应的软件可以使用, 这款软件就是 freerdp.

安装

虽然有apt-get的方式可以直接安装, 但是这个好像不支持密码为空的连接方式, 就直接使用源码进行安装. 

sudo apt-get update -y  # 这步必须要做, 不然可能依赖不能下载完全

sudo apt-get install git cmake cdbs zlib1g-dev libssl-dev libglib2.0-dev libxml2-dev libxext-dev libsystemd-dev libwayland-dev libxkbcommon-dev libasound2-dev libpulse-dev libcups2-dev libpcsclite-dev libavcodec-dev libavutil-dev libavresample-dev libswresample-dev libxkbfile-dev libxinerama-dev libxcursor-dev libxv-dev libxi-dev libxrandr-dev docbook-xsl libcairo2-dev xsltproc libusb-1.0-0-dev -y

sudo git clone git://github.com/FreeRDP/FreeRDP.git --branch 2.2.0

cd FreeRDP

sudo cmake -DTARGET_ARCH=ARM -DWITH_X11=ON

sudo make

sudo make install

使用

安装成功输入xfreerdp出来软件的帮助信息后, 说明安装成功

这个命令的参数有点多, 可以直接参考help, 这里只介绍几个参数的作用

  • /v 主机ip和地址 形式如 /v:192.168.1.100:4489

  • /p 密码

  • /u 用户名

  • /f 全屏

  • /bpp 色彩度指定, 当网络速度不快时, 或者需要降低延迟, 这个值可以指定的比较小

  • /drive 指定挂载主机的目录, 会在远程主机中出现一个虚拟目录, 这个目录用来互相传送文件. 使用方式例如 /drive:Mount,/home/yscl/ 这里的mount指的是挂载的目录名字

  • /w 宽度width

  • /h 高度height

使用demo

xfreerdp /u:JohnDoe /p:Pwd123! /w:1366 /h:768 /v:192.168.1.100:4489

 xfreerdp /v:127.0.0.1 /u:Administrator /p: /f /bpp:8

说明一句, windows的远程访问端口为3389, 当我们连接成功后, 可以按crtl+ alt +enter 退出全屏
